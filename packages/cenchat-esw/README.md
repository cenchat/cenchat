@cenchat/esw
==============================================================================

Packages for [ember-service-worker](https://github.com/dockyard/ember-service-worker)


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.4 or above
* Ember CLI v2.13 or above
* Node.js v8 or above


Installation
------------------------------------------------------------------------------

```
lerna add @cenchat/esw --scope <package> --dev
```
